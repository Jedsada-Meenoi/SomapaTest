# SomapaTest
Show Test Project

## Getting Started

### Backend (Spring Boot)

1. Navigate to excel-demo folder
2. Run the application:
```bash
mvn spring-boot:run
```
3. Server runs on `http://localhost:8080`

### Frontend (Angular)

1. Navigate to sompa-angular folder
2. Install dependencies:
```bash
npm install
```
3. Run the application:
```bash
ng serve
```
4. App runs on `http://localhost:4200`
